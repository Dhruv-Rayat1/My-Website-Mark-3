Time Line

Dhruv Rayat began his coding Journey in 2017 starting with Python. He began by learning the basics: Varaibles, Loops, Conditional Statements etc, he described it as "A great Introduction on how Computers worked". He then quickly started to fall in love with Code.


In late 2020 Dhruv began his Journey into Unity. He then went ahead and started to learn the basics of C# the prefered Langauge for Game Developers. After 3 months of learning he began work on Timecore ©, a game where you can control time, and you have to parcour up stages.


While Still learning C#(Unity) Dhruv decided to learn Vanilla C#, starting with Variables, Loops, API's, etc. He then decided to start testing his skills by creating an AI called Paige. Paige can tell time, Open apps, Tell Jokes, Search the web; and even more! He started development on this AI while working on Pogformer, and is proud to say Paige's code is now a GIT Repo on his Github


After a quick break Dhruv started work on his second Game Pogformer ©, this was a 2D Platfomer game where you had to defeat enimies, jump over blocks and reach the floating green cube to complete the level.


Once Dhruv's game development phase was over, he decided to rebrand himself, he started as a Python Developer, and Finally he decided to become a web Developer, he quickly started getting a hang of HTML, CSS and JS, he then started to get a couple websites under his belt.
